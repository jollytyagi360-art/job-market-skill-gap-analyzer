# AI Job Market Skill Gap Analyzer

This project analyzes 1.6M+ job postings to identify high-demand and low-competition skills in the Indian job market using Python, NLP, and Power BI.

## Project Overview
The system processes large-scale job data, extracts skills from job descriptions, calculates skill demand, and computes a skill gap score to highlight the most valuable skills to learn.

## Features
- Large-scale data cleaning using Python
- NLP-based skill extraction from job descriptions
- Skill demand calculation
- Skill gap scoring
- Interactive Power BI dashboard

## Tech Stack
- Python (Pandas)
- NLP (rule-based)
- Power BI

## Folder Structure
- notebooks/ → Python scripts
- data/ → Processed datasets
- dashboard/ → Power BI dashboard

## Key Insights
Python, SQL, and AWS show the strongest combination of demand and opportunity in the current Indian job market.
